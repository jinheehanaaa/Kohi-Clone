# Installation & Setup
## Vulkan
- Install Vulkan by visiting official website
- ./vulkansdk
- ./vulkansdk samples
- source setup-env.sh
- echo $VULKAN_SDK
- sudo apt install libx11-dev libxkbcommon-x11-dev
